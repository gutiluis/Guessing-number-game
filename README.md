# Number Guessing Game

A simple console-based number guessing game written in Python. The player tries to guess a randomly chosen number within a limited number of attempts.

---

## Features

- Generates a random number between a specified range (default 1–100).
- Prompts the player to guess the number.
- Provides feedback if the guess is too high or too low.
- Tracks the number of attempts.
- Congratulates the player if they guess correctly.
- Ends the game if the player exceeds the maximum number of guesses.

---

## How to Play

1. Run the game in your terminal.
2. The game selects a random number.
3. Input your guess when prompted.
4. The game will tell you if your guess is **too high**, **too low**, or **correct**.
5. You continue guessing until you find the number or run out of attempts.
